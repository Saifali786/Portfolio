# Personal Portfolio

A simple personal portfolio built with React, JSX, Vite, and Tailwind CSS.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Start development server

```bash
npm run dev
```

The app runs on `http://localhost:8080`.

## Build for Production

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```

## Lint

```bash
npm run lint
```
